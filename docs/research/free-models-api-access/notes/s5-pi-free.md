# Source Note: S5

## Metadata
- Source: pi-free GitHub Repository
- Date: 2026
- Type: repo
- Quality: high

## Key Claims
- pi-free is a PI extension that unlocks free and paid AI models from multiple providers
- Free models (no payment required): kilo/, openrouter/, cline/, llm7/
- Freemium (free tier with limits): nvidia/ (1K free req/month), ollama-cloud/ (usage-based), sambanova/ (20-480 RPM free)
- Dynamic API providers (when API key configured): mistral/, groq/, cerebras/, xai/, huggingface/, openrouter/, fastrouter/
- Auth providers: Kilo OAuth, Cline OAuth, NVIDIA API key, Ollama Cloud API key, etc.
- Kilo free tier: 200 requests/hour; Cline: free account
- NVIDIA: 1,000 free requests/month (zero-cost models don't consume credits)
- Ollama Cloud: free tier resets every 5 hours + 7 days
- SambaNova: 20-480 RPM, 400-9600 RPD, no credit card
- Codestral (Experiment plan): 2 req/min, 500K tokens/min, 1B tokens/month, phone verification only
- LLM7: free tier 100 req/hr, 20 req/min, no credit card

## Limitations
- Third-party community extension, not official PI
- Some providers marked "paid" may have free trial credits
- Model availability depends on upstream provider API changes
