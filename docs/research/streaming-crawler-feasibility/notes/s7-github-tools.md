# Source Note: S7 - GitHub: Open-source M3U8/HLS Download Tools

## Metadata
- Source: https://github.com/topics/m3u8-downloader
- Date: 2026 (various)
- Type: Web (GitHub repository listings)
- Quality: High

## Key Claims
- cat-catch (20k stars): Browser resource sniffing extension
- N_m3u8DL-CLI (16k stars): .NET CLI m3u8/HLS/DASH downloader
- mediago (9k stars): Cross-platform desktop + Docker downloader
- N_m3u8DL-RE (8.1k stars): Cross-platform MPD/M3U8/ISM downloader
- Dozens of other tools of varying quality
- All handle the core problem of fetching segments from a manifest and merging them

## Evidence Extracts
- N_m3u8DL-RE: "Cross-Platform, modern and powerful stream downloader for MPD/M3U8/ISM"
- N_m3u8DL-CLI: supports "AES-128-CBC decryption, multi-threading, custom request headers"

## Limitations
- Quality varies widely; many are abandoned
- Most require you to already have the manifest URL
