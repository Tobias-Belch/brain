## Key Notes

- GEKA "Schlauchstück" = the hose-end fitting that creates the GEKA bayonet coupling on a hose
- GEKA "VX-Verschraubung" = threaded union fitting (allows pipe/hose to connect to GEKA socket on a tap/valve)
- The VX-Verschraubung AG G3/4–25mm (1") (art. 12023875) is the adapter that screws a G 3/4" male thread → GEKA 1" socket. This is the critical fitting for connecting each faucet output to a 1" hose.
- DVGW certification = drinking-water approved (Trinkwasser)
- 360° rotating variants available for ergonomic hose connection

## GEKA Standard
- Bayonet quick-connect (claw coupling) — tool-free connect/disconnect
- German DIN 8381 / DVGW W 543
- Sizes: 1/2", 3/4", 1" — 1" (DN 25) is the largest standard size
- All sizes are cross-compatible via size-matched sockets; 1" sockets only accept 1" plugs
