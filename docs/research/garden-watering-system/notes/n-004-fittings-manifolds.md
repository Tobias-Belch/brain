## Key Assessment

### GARDENA 4-Wege-Verteiler (art. 7816157) — €27.99
- Has 4 individual shutoff outlets → perfect for 4 GEKA faucets
- Input connection is **G 3/4"** (not 1") — requires a 1"→3/4" reducer
- GARDENA uses its own quick-coupling system (not GEKA compatible)
- **Not recommended** for GEKA-based system without adapter work

### GARDENA 2-Wege-Ventil für 1"-Gewinde (art. 1621507) — €10.99
- Input: G 1" — matches pump outlet directly
- Only 2 outlets → would need 2 units in series (4 outputs total via 2× T-config)
- GARDENA coupling on output, not GEKA → use VX-Verschraubung adapter

### Recommended Distribution Strategy: Sequential T-fittings (1" brass T-pieces)
For a pure 1"-diameter run with GEKA at each outlet:
- 1× 1" main line from pump (PE hose or galv. steel pipe)
- 3× brass T-piece 1" (Messing T-Stück G1") + 1× end elbow or straight end
- Each branch: 1" ball valve + GEKA VX-Verschraubung + GEKA socket

**Note:** 1" brass T-pieces are a sanitary/installation fitting, not stocked in garden section — must be sourced from Hornbach Bad & Sanitär or online (Cordes-Graeber, Installationsshop, etc.)

## Pipe Material Recommendation
- **PE hose (Polyethylen-Schlauch) 1"** for flexibility and frost resilience
- Alternative: **galvanized steel pipe (Stahlrohr verzinkt) G1"** for rigid outdoor runs
- PE 25 mm OD flexible hose ~€1–2/m (online); 1" galv. steel ~€5–8/m at Hornbach
