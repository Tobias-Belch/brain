## 1. System Overview

```
[TIP GP INOX 5000 pump]
        | G1" outlet
        |
[1" flex PE hose / galv. pipe — indoor run]
        |
[27 cm wall — core-drilled hole Ø50mm + PVC sleeve]
        |
[1" main line outdoor — PE flex hose or galv. pipe]
        |
   ┌────┴────┐
   T-piece   │
   |         │
 [Outlet 1]  │  (repeat T-pieces for outlets 2, 3)
 ball valve  │
 GEKA socket └──► [Outlet 4 at end of line]
```

**Key design principles:**
- Maintain 1" (DN 25 / G1") diameter throughout for minimal flow restriction
- GEKA original brass fittings at every outlet (DVGW certified, drinking-water safe)
- Individual ball valve per outlet for independent shutoff
- Flexible PE hose preferred over rigid pipe for outdoor garden run (frost/movement tolerant)

## 5. Installation Notes

1. **Core drill first** — rent Ø50mm SDS drill from Hornbach. Mark hole center, drill from both sides if possible. Insert PVC sleeve flush with wall surfaces.
2. **Run pipe through sleeve** — use 1" PE flex hose. Seal indoor side with PU foam, outdoor with silicone.
3. **Install T-chain outdoors** — 3× T-pieces with ball valve + GEKA socket on each branch. Final branch uses elbow or straight end.
4. **Thread sealing** — use PTFE tape (3–4 turns) + Fermit Hanf on all threaded connections.
5. **Test at pressure** — before burying or finalizing, run pump and check all joints for leaks at operating pressure.
6. **Frost protection** — drain system in autumn. PE hose and brass fittings tolerate frost better than PVC/plastic.
