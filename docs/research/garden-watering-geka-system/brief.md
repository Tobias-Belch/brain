# Research Brief

## Question
What piping systems and materials are compatible with 1-inch GEKA connectors for a garden watering system, and what specific parts (2x 90° elbows, 30cm pipe, 4-way distributor, 1-inch GEKA-compatible faucets) are needed for each system — with links, Berlin pricing, assembly difficulty, and durability?

## Intended Use
Planning and purchasing decision for a garden watering installation. Pump (TIP GP INOX 5000) has one 1-inch output. Pipe must pass through a 27cm thick wall. Four faucets with GEKA connectors required. Goal: maximize pressure/flow by using maximum pipe/fitting diameters throughout.

## Depth and Checkpoint Mode
- Depth: standard
- Mode: default

## Scope
- In scope: GEKA-compatible 1-inch systems (copper, stainless steel, galvanized steel, PE/PEX, PVC/CPVC, push-fit), specific parts, German/Berlin pricing, assembly effort, longevity
- Out of scope: drip irrigation internals, sprinkler head specs, electrical wiring

## Source Policy
- Preferred source types: web (German retailers, manufacturer sites), docs
- Source distrusts / exclusions: none
- Recency constraints: current products in stock (2024-2026)

## Tool Policy
- Default allowlist: WebFetch
- Run allow overrides: none
- Run disallow overrides: none

## Evidence Threshold
Medium confidence acceptable; cite sources for all price claims. Flag single-sourced prices.

## Defaults and Assumptions
- Depth defaulted to standard (not specified by user)
- Checkpoint mode defaulted to default
- Berlin, Germany pricing scope assumed from user context
- 1 inch = DN25 / G1" throughout
- GEKA = standard garden quick-coupling system (not proprietary brand only)
