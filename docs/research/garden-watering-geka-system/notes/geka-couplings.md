# GEKA Couplings — Confirmed Prices (Amazon.de, Berlin 12619, 2026-05-29)

## What is GEKA?
GEKA = German agricultural/horticultural quick-coupling standard (Bajonettverschluss / claw-coupling).
Interchangeable between brands: VARIOSAN, Poppstar, MegaHaustechnik, Geka GmbH, Agora-Tec, etc.
Connection size for 1" systems: 40mm claw diameter, 25mm bore.
