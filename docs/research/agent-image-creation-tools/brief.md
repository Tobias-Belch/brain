# Research Brief

## Question
Are there any skills, MCPs, or other tools that allow agents to properly use image creation tools, specifically for creating raster or vector images, in the style of cute cartoon/kawaii illustrations (like potted plants with faces, simple line art, pastel colors)? Will agents be capable of creating such images?

## Intended Use
Evaluate current tooling options for enabling AI agents (like opencode/Claude) to generate styled raster and vector images autonomously, and assess feasibility for producing kawaii/cartoon illustration assets.

## Depth and Checkpoint Mode
- Depth: deep
- Mode: default (checkpoints enabled)

## Scope
- In scope: Skills, MCPs, plugins, APIs, SDKs for image generation by AI agents; raster formats (PNG, JPG, WebP); vector formats (SVG); styled illustration capabilities (cartoon, kawaii, flat design); agent-native image generation workflows
- Out of scope: Non-agent image generation tools (standalone apps like Photoshop), image editing/post-processing pipelines (unless agent-driven), social media image posting

## Source Policy
- Preferred source types: web, repos, docs, papers
- Source distrusts / exclusions: marketing-only content without technical detail
- Recency constraints: 2024-2026 preferred; older if still relevant and maintained

## Tool Policy
- Default allowlist: all available
- Run allow overrides: none
- Run disallow overrides: none

## Evidence Threshold
High confidence required for capability claims; moderate for emerging/experimental tools

## Defaults and Assumptions
- User is operating within the opencode ecosystem
- "Agent" refers to LLM-based coding/development agents with tool use
- Style target: simple, flat, kawaii/cute cartoon illustrations with minimal detail
