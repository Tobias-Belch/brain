# Research Brief

## Question
Which click connections for garden watering exist, by which brands? Are there differences in functionality, quality, and price? When using 3/4 inch hoses, which ones should be used?

## Intended Use
Create a compatibility map and buying decision support for garden watering click-connector systems available in Germany, especially for 3/4 inch hoses.

## Depth and Checkpoint Mode
- Depth: deep
- Mode: default

## Scope
- In scope: Garden watering quick/click hose connector systems and relevant threaded tap/hose fittings available in Germany; brand ecosystems; cross-compatibility; 3/4 inch hose use; functionality, quality, and price tradeoffs.
- Out of scope: Industrial hydraulic/pneumatic couplings, permanent irrigation pipe systems, social media sentiment, private retailer data, and unavailable/private marketplaces.

## Source Policy
- Preferred source types: Manufacturer product pages and manuals, retailer listings/prices in Germany/EU, standards-adjacent documentation where available, independent tests/reviews when credible.
- Source distrusts / exclusions: Unsourced marketplace reviews, SEO-only listicles, social media posts, and non-German availability claims unless corroborated.
- Recency constraints: Prefer current product pages and prices accessed during this run. Treat prices as volatile snapshots.

## Tool Policy
- Default allowlist: webfetch, local file read/write within run folder, shell for directory setup/timestamps, task/explore agents for parallel research.
- Run allow overrides: none.
- Run disallow overrides: none.

## Evidence Threshold
Important compatibility and recommendation claims should be corroborated by manufacturer or retailer evidence where possible. Price claims require timestamped retailer/manufacturer source context and should be labeled as volatile.

## Defaults and Assumptions
- User intends to buy/use products in Germany.
- 3/4 inch hose means common German/EU garden hose internal diameter around 19 mm, not necessarily 3/4 inch threaded tap size.
- Default source classes are web and documentation because no local artifacts were supplied.
- Default checkpoint mode is used because `--afk` was not supplied.
