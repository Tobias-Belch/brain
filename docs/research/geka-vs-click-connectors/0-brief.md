# Research Brief

## Question
Should GEKA couplings be used beyond faucet connections in a garden watering setup, compared with common click-connector systems, for three connection tasks: front split, sprinkler connection, and hand sprayer connection?

## Intended Use
Buying decision.

## Depth and Checkpoint Mode
- Depth: deep
- Mode: default

## Scope
- In scope: GEKA-style claw couplings, common garden click connectors, front splitter/distributor connection, sprinkler connection, hand sprayer connection, Germany availability, indicative Germany pricing, setup recommendation, practical tradeoffs for pressure/flow, durability, ergonomics, and compatibility.
- Out of scope: irrigation controller design, pump sizing, underground pipe layout, drip irrigation internals, non-Germany retail availability, professional agricultural quick-coupler systems outside normal garden use unless directly relevant.

## Source Policy
- Preferred source types: manufacturer product/spec pages, German retailer pages, price-comparison pages, prior local garden-watering research in this repository.
- Source distrusts / exclusions: marketplaces and retailer pages are acceptable for pricing but treated as lower authority for technical claims; blocked or JS-only pages are logged but not used for unsupported factual claims.
- Recency constraints: buying/pricing/availability claims should reflect pages fetched during this run on 2026-05-29 where possible. Older or undated technical pages may support stable connector characteristics but not current pricing.

## Tool Policy
- Default allowlist: WebFetch, local file reads/searches, local file writes inside this run folder.
- Run allow overrides: none.
- Run disallow overrides: none.

## Evidence Threshold
Decision-critical claims should be corroborated when practical. Single-source technical or price claims may be used if clearly labeled. Blocked-source and availability gaps must be surfaced in the final recommendation.

## Defaults and Assumptions
- The intended use was captured as buying decision.
- The user asked for deep research; target source budget is 12-30 sources if accessible.
- Germany is the relevant purchasing geography.
- "GEKA" means GEKA-style claw quick couplings, including original GEKA and compatible products, unless a source specifically distinguishes original-branded GEKA.
- "Click connectors" means consumer garden push/click hose connector systems such as GARDENA Original System and compatible alternatives.
- The key purchase decision is not whether GEKA works at all, but where its advantages justify its higher bulk, lower one-hand convenience, or accessory-compatibility friction.
